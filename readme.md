# 4o# 4or

A language interpreter by Yury Lebedev for the language `4or`, implemented with the guidance of [Crafting Interpreters by Robert Nystrom](https://craftinginterpreters.com/)

# Usage
Run REPL
```
cargo r
```

Run web REPL
```
trunk serve --open
```
